Task 4.5: Autonomous Movement Logic Test

Test Objectives

  • Random Direction Changes: Character should randomly change direction while walking
  • Boundary Detection: Character should detect viewport edges
  • Boundary Response: Character should bounce, reverse direction, or change behavior at boundaries

Requirements: 5.1 (viewport containment), 5.2 (boundary response)

Manual Observations

Watch the character for 30-60 seconds and verify:

  1. Character moves autonomously without user input
  2. Character changes direction randomly while walking
  3. Character stays within canvas boundaries (never goes outside)
  4. Character responds to boundaries (bounces, reverses, or changes state)
  5. Character exhibits varied behaviors (walking, jumping, idle)
